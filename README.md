# Python-BugTrackingToolApp
This is a repository to store code that I wish to show to possible employers.
This program is a digital bug tracking tool written in Python. The user can enter input to choose to Add a bug report to the program, or view the bugs currently in the program, or change the status of the bugs that were added, or exit the program which then saves the result to a JSON file. This code makes sure the user does not enter empty input for the bug reports and uses .Lower() so that capitilization is not a problem. A while loop keeps the program running until the user types exit.
unlike C# python can be run without a build command, you run this program by typing python BugTrackingTool.py in the terminal.

This program demonstrates console application development, user input handling, loops and conditional logic, file handling with JSON, and basic debugging practices.
